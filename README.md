# HTML & CSS Project

A simple, clean starter template for building websites with HTML and CSS.

## Project Structure

```
Nikon-Coolpix/
├── index.html       # Main HTML file
├── styles.css       # CSS stylesheet
└── README.md        # This file
```

## Getting Started

1. Open `index.html` in your web browser to view the website
2. Edit `index.html` to modify the content
3. Edit `styles.css` to change the styling

## Features

- Clean, semantic HTML5 structure
- Responsive CSS design
- Mobile-friendly navigation
- Modern styling with flexbox
- Easy to customize

## How to View

Simply open `index.html` in your preferred web browser:
- Double-click the file in File Explorer, or
- Right-click and select "Open with" your browser, or
- Use the Live Server extension in VS Code for live preview

## Customization

- Update the content in `index.html`
- Modify colors, fonts, and layout in `styles.css`
- Add more sections or pages as needed

## Browser Support

Works in all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
